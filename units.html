Css Units

  1. px(pixels) => fixed(not used for responsive design)
  2. rem => relative to the root elements (html) => default 16px
  3. em => relative to parent element.
  4. % => relaive to parent element.
  5. vh => relative to the viewport's height
  6. vw => relative to the viewport's width
